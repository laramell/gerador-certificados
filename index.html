Gerador de Certificados

1
Lista de nomes

Envie um arquivo .txt com os nomes separados por vírgula. Recomendamos até 50 nomes por vez; para listas maiores, divida em etapas.

2
Configurar arquivo

O arquivo deve estar em PNG no tamanho A4 obrigatoriamente

3
Texto do certificado

Digite abaixo o modelo do certificado. Use a letra X no lugar onde o nome do participante deve aparecer.

4
Gerar certificados

Após configurar o certificado e ajustar a posição do texto, clique para gerar. O sistema criará todos os PDFs da lista de nomes e exportará um único arquivo ZIP para download.
